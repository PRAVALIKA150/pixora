# Pixora – AI Powered Cloud Image Sharing Platform

## Overview

Pixora is a cloud-based image sharing platform built on Microsoft Azure. Users can upload images, view images uploaded by others, like and comment on posts, and manage content through a modern web interface.

The project demonstrates how multiple Azure services can work together in a real-world application.

---

## Features

* User Registration and Login
* Image Upload and Storage
* Like and Comment System
* Responsive User Interface
* Azure Cloud Deployment
* Secure Secret Management using Azure Key Vault
* Azure SQL Database Integration
* Azure Blob Storage Integration

---

## Azure Services Used

### Azure App Service

Hosts the Node.js web application.

### Azure Blob Storage

Stores uploaded images securely.

### Azure SQL Database

Stores user information, image metadata, likes, and comments.

### Azure Key Vault

Stores application secrets and connection strings securely.

### Azure AI Vision

Used for image analysis and AI-powered enhancements.

---

## Architecture

User Browser

↓

Azure App Service (Node.js + Express)

↓

Azure Blob Storage (Image Storage)

↓

Azure SQL Database (Metadata, Users, Likes, Comments)

↓

Azure Key Vault (Secrets Management)

↓

Azure AI Vision (Image Analysis)
